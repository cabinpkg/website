poac.pm is a package registry for Poac, a package manager for C++.

### Links

> An official website:<br>
https://poac.pm/

> Documentations:<br>
https://doc.poac.pm/

### For developers

#### Install Node.js dependencies

```bash
$ yarn install
```

#### Start endpoint

```bash
$ yarn dev
```

Now you can visit [`localhost:3000`](http://localhost:3000) from your browser.
